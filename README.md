4. **ngrok**: Tunneling for public access.

## ⚙️ Configuration (`~/.vnc/xstartup`)
The setup uses a forced Cinnamon 2D session to provide the authentic Linux Mint experience:
```bash
export XDG_CURRENT_DESKTOP=X-Cinnamon
export XDG_SESSION_DESKTOP=cinnamon
export CINNAMON_2D=1
dbus-launch --exit-with-session cinnamon-session --session cinnamon2d &
```

## 🌙 Sleep Mode & Lid Closure
